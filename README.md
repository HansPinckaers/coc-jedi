# coc-jedi

coc.nvim wrapper for the jedi-language-server for Python

## Install

`:CocInstall coc-jedi`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-jedi-keymap)`

## Lists

`:CocList demo_list`
